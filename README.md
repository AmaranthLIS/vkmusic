### Описание

Десктопное, кроссплатформенное приложение для выгрузки аудио библиотеки из ВК.

![Скриншот](docs/screen.png "Скриншот")


### Запуск

#### Linux

```$ chmod +x vkmusic-*.jar```

```$ ./vkmusic-*.jar```


#### Linux / Windows / MacOS

```$ java -jar vkmusic-*.jar```
